Transform a CubeGen Model into Code according to your own Template.

Introduction

Command line

cubegen.exe <model> <template> <code> <arguments>
perl cubegen.pl <model> <template> <code> <arguments>
- <model>: Imported Cube Model textfile
- <template>: Importede CubeGen Template Textfile
- <code>: Textfile to generate
- <argument>: Parameters used in template

Case1

CubeModel

=BUSINESS_OBJECT_TYPE[BOT-000000100000]:CUSTOMERS|customer_files;
=BUSINESS_OBJECT_TYPE[BOT-000000100001]:ORDER|order_directory;
=BUSINESS_OBJECT_TYPE[BOT-000000100002]:PRODUCT|product_definitions;

Iteration

Template

List:[[LOOP,BUSINESS_OBJECT_TYPE]]
-	<<BUSINESS_OBJECT_TYPE:U>>[[ENDLOOP,BUSINESS_OBJECT_TYPE]]

Code

List:
-	CUSTOMERS
-	ORDER
-	PRODUCT

Replacement

Template

[[LOOP,BUSINESS_OBJECT_TYPE]]
Uppercase: <<BUSINESS_OBJECT_TYPE:U>>
Lowercase: <<BUSINESS_OBJECT_TYPE:L>>
Number: <<BUSINESS_OBJECT_TYPE:N>>
Subnumber: <<BUSINESS_OBJECT_TYPE:S>>
CamelCase: <<BUSINESS_OBJECT_TYPE:C>>
ID: <<BUSINESS_OBJECT_TYPE:I>>
[[ENDLOOP,BUSINESS_OBJECT_TYPE]]
